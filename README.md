# Reboot 💪

Piano di allenamento giornaliero interattivo — una timeline di 24 ore con esercizi fisici, tempi di riposo e consigli per uno stile di vita attivo.

**Live site:** [simonestella.github.io/reboot](https://simonestella.github.io/reboot)  
**Portfolio:** [simonestella.github.io/portfolio](https://simonestella.github.io/portfolio)

## Funzionalità

- Timeline interattiva di 24 ore con esercizi fisici
- Schede dettagliate per ogni esercizio (descrizione, serie, ripetizioni, esecuzione)
- Consigli quotidiani per la salute e il benessere
- Dark / Light mode
- Italiano / English
- Deploy automatico su GitHub Pages

## Stack

- [Next.js](https://nextjs.org) (App Router, static export)
- [Tailwind CSS v4](https://tailwindcss.com)
- [Framer Motion](https://www.framer.com/motion/)
- [next-themes](https://github.com/pacocoursey/next-themes)

## Sviluppo locale

```bash
npm install
npm run dev
```

Apri [http://localhost:3000](http://localhost:3000).

## Deploy

Il sito viene deployato automaticamente su GitHub Pages ad ogni push su `main`.

Per abilitare GitHub Pages sul repository:
1. Vai su **Settings → Pages**
2. Source: **GitHub Actions**
3. Fai un push su `main` per avviare il primo deploy

Il sito sarà disponibile su `https://simonestella.github.io/reboot`.
